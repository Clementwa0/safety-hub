# HSE Hub

A modern PPE e-commerce platform built with Next.js 15, TypeScript, Tailwind CSS, and Supabase.

## Features

- PPE Product Catalog
- Product Categories
- Search & Filtering
- Shopping Cart
- WhatsApp Checkout
- Quotation Requests
- Invoice Generation
- Admin Dashboard
- Inventory Management
- Customer Management
- Order Tracking
- Analytics & Reports

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- Supabase
- Framer Motion
- Lucide React
